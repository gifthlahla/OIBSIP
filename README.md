## How to Run Any Task

### Prerequisites
- Java JDK 11 or higher installed
- Terminal or command prompt

### Steps for Task 1 and Task 2 (Single File)
1. Navigate to the task folder:
   ```bash
   cd Task1_NumberGuessing
   ```
2. Compile:
   ```bash
   javac NumberGuessingGame.java
   ```
3. Run:
   ```bash
   java NumberGuessingGame
   ```

### Steps for Task 4 (Multiple Files)
   1. Navigate to the task folder:
   ```bash
   cd Task4_OnlineExam
   ```
2. Compile all Java Files:
   ```bash
   javac *.java
   ```
3. Run:
   ```bash
   java OnlineExamSystem
   ```
